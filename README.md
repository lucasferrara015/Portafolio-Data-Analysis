# 📊 Portafolio de Proyectos - Análisis de Datos

¡Bienvenido/a a mi portafolio profesional de Análisis de Datos! En este espacio comparto proyectos prácticos donde resuelvo problemas de negocio y comportamiento social utilizando datos del mundo real.

Mi enfoque abarca desde la captura e ingeniería de datos hasta el análisis estadístico avanzado, el modelado y la creación de narrativas visuales (*Data Storytelling*) eficaces para la toma de decisiones estratégicas.

## 📂 Índice de Proyectos
A continuación se presentan los proyectos desarrollados, organizados analíticamente por competencias técnicas según el ciclo de vida de los datos:

### 1. Limpieza y Transformación de Datos (Data Wrangling)
* **[Proyecto 1: Curación y Optimización del Dataset de Netflix](./01%20-%20Data%20Wrangling)**
  * **Objetivo:** Tomar un dataset masivo e incompleto y transformarlo en un recurso estructurado y optimizado (0% nulos).
  * **Tecnologías:** Python, Pandas, NumPy.
  * **Claves:** Imputación avanzada de nulos, parsing de fechas (`datetime64`), ingeniería de características (*Feature Engineering*) y detección estadística de outliers mediante el método IQR.

### 2. Análisis Exploratorio de Datos (EDA) & Comportamiento Humano
* **[Proyecto 2: Lanzamientos de Misiones Espaciales](./02%20-%20Space%20Missions%20EDA)**
  * **Objetivo:** Identificar patrones históricos, organizaciones dominantes y variaciones de costos temporales en la carrera espacial.
  * **Tecnologías:** Python, Pandas, Matplotlib.
* **[Proyecto 3: Communication in the Digital Era (Social EDA)](./03-communication-digital-era-eda)**
  * **Objetivo:** Investigar las dinámicas de conectividad modernas, analizando cómo las variables socioeconómicas y geográficas moldean el uso de canales digitales y patrones de interacción en la sociedad actual.
  * **Tecnologías:** Python, Pandas, Seaborn, Estadística Descriptiva.
  * **Claves:** Segmentación de usuarios, análisis de frecuencia de uso y visualización del impacto tecnológico en relaciones interpersonales.

### 3. Visualización de Datos Avanzada & Inteligencia de Negocios (BI)
* **[Proyecto 4: E-commerce Market Analysis & Pricing Strategy (Amazon)](./04%20-Dashboard%20de%20Ecommerce)**
  * **Objetivo:** Analizar las estructuras comerciales y de fijación de precios en más de **466,000 productos**, validando científicamente el impacto del costo sobre la satisfacción del cliente.
  * **Tecnologías:** Python (Pandas, Seaborn, Matplotlib).
  * **Claves:** Ingesta y unificación vertical de 104 archivos CSV, aislamiento y remediación de anomalías de calidad de datos (*Outliers* de 9.9 mil millones), cálculo del Coeficiente de Correlación de Pearson (`-0.0043`), visualización en cuadrantes ejecutivos de alta resolución (300 DPI) con mapas de dispersión, líneas de regresión y diagramas de caja (*Boxplots*).

### 4. Ciencia de Datos (Modelado Predictivo)
* **[Proyecto 5: Predicción de Precios de Vuelos](https://github.com/lucasferrara015/Portafolio-Data-Analysis) 
  * **Objetivo:** Desarrollar un modelo de machine learning (Regresión) para estimar costos tarifarios aéreos basados en estacionalidad y características operativas.
  * **Tecnologías:** Python, Scikit-Learn.

### 5. Ingeniería de Datos (Modelado & Pipelines ETL)
* **[Proyecto 6: Pipeline ETL para Monitoreo de Tendencias Musicales Globales (Last.fm API)](./06%20-%20Ingenieria%20de%20Datos%20Music)**
  * **Objetivo:** Diseñar y construir un proceso ETL (Extracción, Transformación y Carga) de extremo a extremo que interactúa con la API de Last.fm para recopilar de forma estructurada los rankings globales de reproducción, modelando un Data Warehouse histórico para identificar patrones en la industria musical.
  * **Modelo de Datos:** Esquema en Estrella (Tablas de Hechos de conteos diarios de reproducciones y Dimensiones independientes para Canciones, Artistas y Tiempo).
  * **Tecnologías Clave:** Python (Requests, Pandas, SQLAlchemy), SQL (MySQL / MySQL Workbench).
