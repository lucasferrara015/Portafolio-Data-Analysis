# 📊 Portafolio de Proyectos - Análisis de Datos

¡Bienvenido/a a mi portafolio profesional de Análisis de Datos! En este espacio comparto proyectos prácticos donde resuelvo problemas de negocio utilizando datos del mundo real. 

Mi enfoque abarca desde la captura e ingeniería de datos hasta el análisis estadístico y la creación de dashboards interactivos para la toma de decisiones.

## 🛠️ Tecnologías Globales
* **Lenguajes:** Python (Pandas, NumPy, Matplotlib, Seaborn), SQL
* **Herramientas de BI:** Power BI / Tableau / Looker Studio
* **Entornos:** Google Colab, Jupyter Notebooks, Git/GitHub

## 📂 Índice de Proyectos

A continuación, se presentan los proyectos desarrollados, organizados por competencias técnicas según el ciclo de vida de los datos:

### 1. Limpieza y Transformación de Datos (Data Wrangling)
* **[Proyecto 1: Curación y Optimización del Dataset de Netflix](./01-Limpieza-Transformacion-Netflix/)**
    * **Objetivo:** Tomar un dataset masivo e incompleto y transformarlo en un recurso estructurado y optimizado (0% nulos).
    * **Tecnologías:** Python, Pandas, NumPy.
    * **Claves:** Imputación avanzada de nulos, parsing de fechas (`datetime64`), ingeniería de características (*Feature Engineering*) y detección estadística de outliers mediante el método IQR.

### 2. Análisis Exploratorio de Datos (EDA)
* **[Proyecto 2: Lanzamientos de Misiones Espaciales (Próximamente)](./02-Analisis-Misiones-Espaciales/)**
    * **Objetivo:** Identificar patrones históricos, organizaciones dominantes y variaciones de costos temporales en la carrera espacial.

### 3. Visualización de Datos & Business Intelligence
* **[Proyecto 3: Dashboard de Ecommerce (Próximamente)]**
    * **Objetivo:** Diseñar un cuadro de mando interactivo para analizar KPIs de ventas y comportamiento del consumidor.

### 4. Ciencia de Datos (Modelado Predictivo)
* **[Proyecto 4: Predicción de Precios de Vuelos (Próximamente)]**
    * **Objetivo:** Desarrollar un modelo de machine learning (Regresión) para estimar costos tarifarios aéreos.

### 5. Ingeniería de Datos (Automatización)
* **[Proyecto 5: Pipeline ETL con la API de Spotify (Próximamente)]**
    * **Objetivo:** Automatizar la extracción diaria de datos musicales personales creando tablas de hechos y dimensiones.
